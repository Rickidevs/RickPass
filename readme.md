# 🔒 Rickpass.xyz

Bu layihə, istifadəçilərin şifrələrinin təhlükəsizliyini yoxlamağa yönəlmiş bir veb tətbiqidir. Tətbiq, istifadəçidən alınan məlumatları istifadə edərək, şifrə sızıntısı olub-olmamasını müəyyən etmək üçün müxtəlif alqoritmlər və verilənlər bazaları ilə işləyir.

## Xüsusiyyətlər

- **👤 İstifadəçi Girişi:** İstifadəçilər, şifrələrini daxil edərək yoxlaya bilərlər.
- **🔍 Şifrə Nəzarəti:** Tətbiq, daxil edilən şifrələrin sızdırılıb-sızdırılmadığını yoxlayır.
- **📊 Nəticə Göstərimi:** İstifadəçiyə şifrələrinin təhlükəsiz olub-olmadığına dair aydın bir geribildirim təqdim edir.

## Təhlükəsizlik Məlumatları

- Tətbiq, istifadəçi məlumatlarını qeyd etmir və ya üçüncü tərəflərlə paylaşmır. 
- İstifadəçi şifrələri tamamilə anonim şəkildə yoxlanılır və heç bir log saxlanılmır.
